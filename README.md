# GitHub MCP Workflow Exercise

_Learn AI-assisted development with GitHub Copilot and MCP in less than an hour._

## Welcome

- **Who is this for**: Developers at any experience level looking to learn AI-assisted development workflows.
- **What you'll learn**: How to use GitHub Copilot with MCP tools to investigate bugs, create branches, fix code, and manage Pull Requests.
- **What you'll build**: You will fix a broken World Clock web application using AI-guided workflows.
- **Prerequisites**:
  - Familiarity with [VS Code](https://code.visualstudio.com/)
  - [GitHub Copilot](https://github.com/features/copilot) subscription
  - [GitHub MCP Server](https://github.com/github/github-mcp-server) configured in VS Code
  - Basic Git knowledge (commit, push, branch)
- **How long**: This exercise takes less than one hour to complete.

In this exercise, you will:

1. Use GitHub Copilot to investigate and diagnose a CSS styling bug.
2. Create a hotfix branch and fix the code with AI assistance.
3. Create a Pull Request using GitHub MCP tools.
4. Complete a full DevOps workflow from VS Code.

### How to start this exercise

Simply copy the exercise to your account, then give your favorite Octocat (Mona) **about 20 seconds** to prepare the first lesson, then **refresh the page**.
